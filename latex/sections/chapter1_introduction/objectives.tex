\subsection{Objectives}

The main goal of this project is to build a reliable academic certificate verification platform using blockchain and NFT technology. By doing this, we aim to make checking college degrees much safer, faster, and completely transparent. To achieve this, the project focuses on several specific goals:
\begin{itemize}
    \item Issue unique, non-transferable NFTs to graduates that serve as permanent digital certificates.

    \item Build a secure checking tool that uses cryptographic hashing to instantly prove whether a certificate is authentic or has been altered.

    \item Use the permanent nature of blockchain records to stop degree fraud, text forgery, and unauthorized changes.

    \item Cut down our reliance on slow, manual paperwork and centralized databases by setting up a decentralized system.

    \item Make the verification process simple and clear for schools, companies, and any outside organizations checking credentials.

    \item Deploy an AI-powered assistant to guide users through the platform, explain verification reports, and make the system easier to interact with.

    \item Test and analyze how well this NFT-based platform holds up in real-world academic scenarios regarding its safety, speed, and overall practicality.
\end{itemize}
