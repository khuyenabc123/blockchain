\subsection{Database Diagram}

The proposed system uses both blockchain storage and a traditional database. The blockchain is responsible for storing certificate ownership and verification information, while MongoDB manages application-level data such as student records, certificate records. This hybrid approach improves efficiency by keeping frequently updated information in the database while preserving certificate integrity on the blockchain.
